# Commish Bot

Malevolently Robotting For Life.

### Installation

```bash
make install
```

### Deploying

```bash
make deploy
```

### Testing

There's a test bot set up. You can run

```bash
make configure-test
```

to get the remote set up. Then, assuming you're a collaborator you can run

```bash
make deploy-test
```

to get your current changes up on the test bot.
